#2.3
const는 constant(상수)라는 뜻으로 값이 바뀔 수 없음 (값을 업데이트 할 수 없음)
let으로 변수 선언하면 값을 업데이트할 수 있음 (let 변수이름 = 변수값; 으로 변수 선언)
let은 새로운 것을 생성할 때 사용하는 것 (let으로 선언한 변수 업데이트시 let은 안 붙여도 됨)
*대부분 const를 사용하고 때에 따라 let을 사용함!
*var은 쓰지 말 것!

#2.4
컴퓨터에서 0은 꺼져있다는 뜻, 1은 켜져있다는 뜻
boolean: true or false
null: 그 변수에 아무것도 없음 / 변수가 비어있다기보단 아무것도 없음으로 채워진 것 / 자연적으로 발생되지 않음 / 변수 안에 아무것도 없다는 것을 확실히 하기 위해 씀
undefined: 변수를 만들었지만 정의하진 않음 / 컴퓨터 메모리 안에 존재해 공간은 있지만 값이 들어가지 않음
*true: 켜져있음, false: 꺼져있음, null: 비어있음
